# Food Delivery Data Analysis

## Project Overview
This project analyzes a food delivery dataset to understand customer behavior, restaurant performance, and revenue trends. The data is aggregated from three different sources (CSV, JSON, and SQL) to create a unified dataset for analysis.

## Files Description
- **`food_delivery_analysis.ipynb`**: Jupyter Notebook containing the data loading, merging, and analysis code.
- **`orders.csv`**: Transactional data of food orders.
- **`users.json`**: Customer demographic and membership data.
- **`restaurants.sql`**: SQL dump containing restaurant details (cuisine, ratings).
- **`final_food_delivery_dataset.csv`**: The processed and cleaned final dataset.

## Tech Stack
- **Python**
- **Pandas** (Data Manipulation)
- **JSON & Re** (Data Parsing)

## Key Insights
- **Top Revenue City:** Chennai leads in revenue from Gold members.
- **Highest AOV Cuisine:** Mexican cuisine has the highest average order value.
- **Gold Member Impact:** ~50% of all orders come from Gold members.
